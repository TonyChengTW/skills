# Skills

OpenCode skills repository for Linux system troubleshooting and Proxmox VE cluster diagnostics.

## Skills

| Skill | Description |
|-------|-------------|
| `linux-precision-triage` | Linux system troubleshooting (Ubuntu 24.04+) - token-efficient diagnostics |
| `pve-cluster-doctor` | Proxmox VE cluster diagnostics and maintenance |

## Slash Commands

### Linux Precision Triage
- `/linux-triage` - System health snapshot
- `/linux-net-audit` - Network stack scan
- `/linux-storage-audit` - Storage pool diagnostics
- `/linux-cluster-audit` - Cluster quorum check

### Proxmox Cluster Doctor
- `/pve-vm-stuck` - VM/LXC rescue
- `/pve-health` - Cluster health snapshot
- `/pve-storage-audit` - Storage backend diagnostics
- `/pve-quorum-fix` - Quorum emergency repair
